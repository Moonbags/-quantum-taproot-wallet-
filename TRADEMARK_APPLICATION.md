## USPTO TEAS Plus Application Process

### Step 1: Access USPTO System

1. Visit https://www.uspto.gov/trademarks/apply
2. Select "TEAS Plus" application form
3. Create USPTO.gov account if needed

### Step 2: Complete Trademark Form

#### Applicant Information
- Enter owner name and address
- Specify entity type (individual/corporation/LLC)
- Provide contact email and phone

#### Mark Information
- **Mark Type**: Standard Characters
- **Mark Literal Element**: sha256sol
- **Mark Statement**: The mark consists of standard characters without claim to any particular font, style, size, or color

#### Goods/Services Classification
- **Class**: 009
- **Goods Description**: "Downloadable computer software for cryptocurrency wallets; Downloadable computer software for Bitcoin wallet management; Downloadable computer software featuring quantum-resistant cryptographic security for digital asset management"
- **First Use Date**: December 24, 2025
- **First Use in Commerce Date**: December 24, 2025

### Step 3: Upload Specimen

#### Specimen Requirements (TEAS Plus)
Must show the mark as actually used in commerce:

**Acceptable Specimens**:
1. Screenshot of software showing "sha256sol" mark
2. Product packaging or labels
3. Web page showing downloadable software with mark
4. App store listing with mark

**Recommended Specimen**:
Screenshot of GitHub repository README showing:
- "sha256sol" mark prominently displayed
- Association with software/project
- Public availability (commerce)

#### Technical Requirements
- Format: JPG or PDF
- Max Size: 5 MB
- Resolution: Clear and legible
- Shows: Mark in actual use with goods

### Step 4: Provide Additional Information

#### Attorney Information (if applicable)
- Not required for self-filing
- Can add later if needed

#### Correspondence Information
- Email address for official communications
- Must monitor this email regularly

#### Authorized Signatory
- Name of person signing application
- Title/relationship to applicant
- Signature (electronic)

### Step 5: Declaration

Sign declaration certifying:
- ✅ Applicant is owner of the mark
- ✅ Mark is in use in commerce
- ✅ No other person has right to use mark
- ✅ Facts stated are true and accurate
- ✅ Understanding of penalties for false statements

### Step 6: Pay Filing Fee

- **TEAS Plus Fee**: $250 per class
- **Classes**: 1 (Class 009 only)
- **Total Fee**: $250
- **Payment Methods**: Credit card, debit card, ACH

### Step 7: Review and Submit

1. Review all entered information carefully
2. Verify specimen is correct and acceptable
3. Confirm goods/services description is accurate
4. Click "Submit" to file application

### Step 8: Receive Confirmation

You will receive:
1. **Immediate**: Email confirmation of filing
2. **Serial Number**: Assigned to your application
3. **Filing Date**: Date of submission
4. **Filing Receipt**: Within 1-2 business days

- Geographic significance

If you receive an office action:

2. Deadline: 6 months from issue date
3. Address all points raised
4. Amend goods/services if requested
5. Provide arguments or evidence
6. File response through TEAS

## Trademark Usage Guidelines

### Proper Use

#### As an Adjective
✅ "sha256sol cryptocurrency wallet software"  
✅ "The sha256sol wallet provides quantum resistance"  
✅ "Download sha256sol software from our repository"

#### With Generic Terms
✅ "sha256sol® software"  
✅ "sha256sol™ wallet"  
✅ "The sha256sol brand"

#### Consistent Form
✅ Always use same spelling: "sha256sol"  
✅ All lowercase (as filed)  
✅ No spaces or hyphens

### Improper Use

#### As a Noun (Avoid)
❌ "Download the sha256sol"  
❌ "This sha256sol is quantum-resistant"

#### As a Verb (Avoid)
❌ "sha256sol your Bitcoin"  
❌ "I sha256sol'd my wallet"

#### Modified Form (Avoid)
❌ "SHA256SOL" (different capitalization)  
❌ "sha-256-sol" (added hyphens)  
❌ "sha 256 sol" (added spaces)

### Marking Requirements

Once registered, use ® symbol:
- "sha256sol®"

Before registration, use ™ symbol:
- "sha256sol™"
